This is my first project: a community library website designed to make books more accessible to readers in my area.

The platform allows users to browse available books, check borrowing status, and learn about upcoming library events.

It also highlights featured authors, community reading programs, and provides digital resources for students.

The goal of the project is to combine technology with community service, creating a space where knowledge is shared freely.
